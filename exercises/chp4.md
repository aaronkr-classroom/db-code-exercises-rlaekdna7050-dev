## 퍼즐

**가로**

- 관계를 맺고 있는 두 개체 타입에서 한 개체 타입의 일부 개체 인스턴스만 다른 개체 타입의 개체 인스턴스와 연관되는 것 :부분참여

- ER 모델의 기본 요소 가운데 개체나 관계의 특성을 나타내는 요소 :속성

- 개체의 이름과 개체를 구성하는 속성들을 정의한 것 :개체타입

- 하나의 개체 타입이 동일한 관계 타입에 두 번 참여하는 것 :순환관계

- 각 속성이 가질 수 있는 값의 범위 :도메인

- ER 모델의 기본 요소 가운데 DB가 표현하려고 하는 유·무형의 정보 대상으로서, 
   현실 세계에 존재하면서 서로 구별할 수 있는 요소 :개체

**세로**

- ER 모델을 나타내는 그래픽 방식의 표현 방법 :ER다이어그램

- 관계에 참여하는 하나의 개체에 대해 다른 개체가 몇 개 대응하는지를 나타내는 것 :카디널리티

- 특정 개체에 대해 여러 개의 값을 가질 수 있는 속성 :다중치속성

- 자체적으로 개체를 식별할 수 있는 기본키 속성을 갖지 못해서 독자적으로는 존재할 수 없는 개체 :약한개체

- 관계로 연결된 개체 타입의 수 :차수

- ER 모델의 기본 요소 가운데 개체 간의 의미 있는 연결 또는 연관성을 표현하는 요소 :관계

3. 다음은 쇼핑몰 DB의 회원 개체에 필요한 속성들이다. 각 속성의 종류가 잘못 연결된 것은? 답에 대한 이유도 설명하시오. 2
  
  (1. VIP여부 - 유도 속성
  (2. 휴대폰번호 - 상수 속성
  (3. 회원_id - 기본커속성
  (4. 취미 - 다중치속성
  (5. 직업 -널속성

이유 : 상수 속성이란 시간이 지나도 값이 변하지 않는 속성인데, 휴대폰번호는 값이
변할 수도 있으므로 가변 속성에 해당한다. VIP여부는 구매금액과 구매일수 등에 의해 결정되므로 유도 속성에 해당한다.

4. 어떤 건물을 어떤 건축가가 건축했는지를 표현히는 ER 모댈의 구성 요소 가운데 성격이 다른 것은? 탑에 대한 이유도 설명하시오. 2
  
  (1. 경력
  (2. 가격
  (3. 건축
  (4. 위치

이유 : 경력은 건축가의 속성이고, 가격과 위치는 건물의 속성인데, 건축은 어떤 건축가
가 어떤 건물을 건축한 것인지를 나타내는 것으로, 건축가 개체 타입과 건물 개체 타입
간의 관계(‘건축하다’)에 해당하므로 다른 구성요소들과 성격이 다르다.

5. 다음은 동아리 DB에서 회원 개체 타입과 행사 개체 타입 간의 참여하다 관계를 표현한 것이다. 적합한 관계 유형을 [________] 안에 쓰시오 (스라이드나 책 117쪽 보세요).

정답 : m:n (다 대 다) 유형
한 명의 회원이 여러 행사에 참여할 수 있고, 한 행사에 여러 회원이 참여할 수 있으므로 적합한 관계 유형은 m:n이다.

6. 디음은 경매 사이트 DB에서 작품 개체 타입과 입찰자 개체 타입 간의 낙찰되다 관계를 표현한 것이다. 적합한 관계 유형을  [________] 안에 쓰시오 (스라이드나 책 117쪽 보세요).

정답 : n:1 (다 대 일) 유형
하나의 작품은 한 명의 입찰자에게 낙찰되고, 한 입찰자는 여러 작품을 낙찰받을 수 있으므로 적합한 관계 유형은 n:1이다.
